主窗口（左右两栏）

持久、慢节奏浏览，是「我的数据 / 配置」。三分区共用左导航 + 条目卡 + 详情抽屉。

收藏视图主动留存

我认可、值得复习的语言素材。可分类、可溯源到原文。每条带「重新解析」。

历史里的好内容 →「转存」→ 收藏
历史视图自动记录

查过的每次解析自动入库，时间倒序，可清空。是收藏的「来源池」。

设置视图gate + 偏好

AI 接入 + 权限构成 setup gate；动作优先级 / 默认动作等偏好会下行影响浮层。

瞬时态 · 浮层

浮层（盖在别的 App 上）

临时、轻、即用即走。同一浮层的两种触发态，共享动作条与结果面板。

选取视图有选区

选中文字即唤起。动作条：翻译 / 语法 / 改写 / 例句 / 收藏 / 发音。英文默认翻译。

顶部 pill 互切
输入视图无选区

输入模式，服务改写与自然英文表达（非通用问答）。中文 / 混合默认改写。

两态共用：动作条 · 结果面板 · 收藏/复制。即你已实现的 relaunch 面板形态。

01 两种性质别混

浮层轻、被触发、即走；主窗口重、被打开、慢浏览。别把设置塞进浮层 tab，也别把收藏塞进划词浮层。

02 三分区同构复用

收藏 / 历史 / 设置同在主窗口，共用左右两栏 + 条目卡 + 详情抽屉。历史与收藏只是数据源和排序不同。

03 relaunch 是关键回路

「重新解析」连接两层，已经做出来了。把它确立为正式的跨层机制，串起沉淀 → 再学习的闭环。
